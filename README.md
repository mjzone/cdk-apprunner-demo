# Initial Steps

- npm i -g aws-cdk@1
- clone the repo
- You should install AWS CLI and configure IAM credentials
- npm i
- cd cdk && npm i
- create github repo
- add new remote url
- commit and push code
- create apprunner github connection
- fill placeholders
- commit and push code
- inside the cdk folder -- npm run build
- cdk bootstrap
- cdk deploy
